# F1 ERS AI Strategy Optimizer

An AI-powered application to optimize Energy Recovery System (ERS) strategy for Formula 1 cars.
This project uses:
- **FastF1** for telemetry data
- **Genetic Algorithms** for optimization
- **Flask** for the web interface

## Deployment
This Space is configured to run using Docker.
The backend runs a Flask server exposed on port 7860.
